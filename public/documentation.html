Building a server-side component of a movies web application. This web application will provide access to information about different movies, directors, and genres. Users will be able to sign up,update their personal information, and create a list of their favorite movies.

Number Request URL HTTP Method Request body data format Response body data format
1 Get a list of all movies /movies GET None A JSON object holding data about all the movies
2 Get data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie,containing an title, description,director, and genre property.
"Genre": {
"Name": "Sci-Fi",
"Description": " "The Saga ends in this thrilling movie packed with action and redemption stories."
}
3 Return genre by name /movies/[genre]/[genreName] GET None A JSON object holding data about the genre class
Example:{
{
"Name": "Sci-Fi",
"Description": "The Saga ends in this thrilling movie packed with action and redemption stories."
}
4 Get data about director by name /movies/[directors]/[directorName] GET None A JSON object holding data about a director.Example
{
"name": "George Lucas",
"Birth Year": "1944",
"Bio": "George Walton Lucas Jr. is an American film director, producer, screenwriter, and entrepreneur. Lucas is best known for creating the Star Wars and Indiana Jones franchises and founding Lucasfilm, LucasArts, and Industrial Light & Magic."
}
